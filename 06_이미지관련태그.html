img 태그

        -웹 문서에서 사진이나 그림같은 이미지를 삽입하는 용도의 태그

        [img 태그의 3가지 속성]
        (속성: 시작 태그에 작성하는 내용)

        1. src 속성 : 삽입할 이미지의 경로를 작성하는 속성
                    (파일 위치, 인터넷 주소 등)
        2. width / height 속성 : 너비/ 높이를 지정하는 속성(이미지 크기)
        3. alt(대안) 속성 : 이미지 설명을 작성하는 속성
                            ->이미지가 출력되지 않은 경우 화면에 출력됨(X박스..이미지 없는 경우에 화면에 나옴)
                            +웹 접근성(스크린 리더가 이미지 설명을 읽어줌)

img 예시

절대 경로로 이미지 출력하기

절대 경로: 변하지 않는 기준점 으로부터 목표 까지의 경로(예시: 서울시 중구 남대문로 120 청계 그레이츠 2층)

-윈도우 기준 : C드라이브, D드라이브가 윈도우의 절대 경로 예시

상대 경로로 이미지 출력하기

상대 경로: 기준점이 그때그때 바뀌는 것 (현재 파일/페이치 위치를 기준으로 하여 목표까지의 경로를 나타내는 것)
(예시: 오른쪽 뒤에 뒤에 뒤에 분한테 전달하기)

고정 크기 단위(px,픽셀): 항상 고정된 크기를 지님

가변 크기 단위(%): 화면 또는 영역(div)에 대한 상대적인 크기 단위

고정 + 가변 크기

고정 크기로 설정된 요소(div 태그) 내부에서 
일정한 크기의 이미지 또는 요소를 출력하고 싶을 때 
가변 크기(%)를 이용해서 지정(div 크기의 몇 %)
