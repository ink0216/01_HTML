style 태그 내용 적용 확인

block 형식

    block=레고쌓아 올리는 것
    -화면(공간)의 수직길이를 가로로 분할(세로 화면을 가로로 잘라서 아파트처럼 층(행)을 나눔)
    -화면의 한 층(행)을 모두 차지함
    -width, height와 같은 크기 지정 속성 사용 가능
    -block 형식의 태그:h1~h6(제목 태그), p/pre(문단 태그), hr(수평선 태그), 
    div(영역 분할 태그),시맨틱 태그(div 태그에 이름 넣은 태그) : 해당 한 줄을 다 차지하는 태그들

h3는 block형식 태그입니다

  block형식은 작성된 내용이 짧더라도 
  한 줄(한 층)을 모두 차지하려는 성질 때문에 
  다음 태그 내용은 다음 줄에 작성됩니다
첫 번째 div
두 번째 div
세 번째 div

inline 형식

inline: 줄 안에 쓴 만큼 나눠짐 (수평으로 자름)
-화면을 수평으로 분할
(가로 화면을 세로로 잘라서 여러 아파트 호수로 나누듯 열로 나눔)
-한 층(행)을 모두 차지하지 않음
  -> 모두 차지하지 않고, 내부에 작성된 내용 크기만큼만 차지함
  ->별도로 크기를 지정할 수 없음(내용 크기만큼만 할 수 있음)
    width, height 와 같은 크기 지정 속성 사용 불가!!!!!!
  
-inline 형식의 태그 
b(bold, 굵게), i(기울이기), mark(형광펜 칠하기), span(영역 분할), strong,abbr,em, 등등..

inline 형식은 행을 분할하는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 span 두 번째 span 세 번째 span
첫 번째 span두 번째 span세 번째 span

영역 지정 방식의 차이점

  • block 형식: 한 줄이 항상 직사각형 박스 형태로 영역이 지정됨
  • inline 형식: 내부에 작성된 내용(content) 단위로 영역을 지정함

inline-block 형식

-inline의 특징 중 수평분할(옆으로 작성하는 특징, 줄을 세로로 자름, 열 나눔)
-block의 특징 중 크기 지정 가능 성질
-위의 두 가지 성질이 섞여 있는 것임
대표적인 태그: img 태그(옆으로 나열됐는데 크기도 지정 가능했음), iframe 태그
*iframe 태그(frame:틀, 액자)
-웹 문서(html) 내부에 
또 다른 웹 문서를 추가하는 태그 (옆에 액자 만들어서 다른 페이지 보이도록)
