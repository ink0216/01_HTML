a(anchor : 연결, 고정) 태그

-현재 HTML 문서와 다른 HTML 문서를 연결 시켜 
이를 통해 페이지를 이동할 수 있게 만드는 태그들
-a태그로 감싸진 글/내용 ==하이퍼텍스트(==하이퍼링크)
-하이퍼 텍스트는 다른 문서와 구분되는 HTML만의 핵심 기능

a 태그 테스트

a 태그의 속성 2가지

 
- href(HyperText REF): 연결할 주소를 지정하는 속성
- target = "_blank" : 새 탭에서 열기

페이지 내부에서 점프하기

-한 페이지 내부에서 
 지정된 id 속성을 가진 요소(태그)로 이동하는 것

*id(식별자, 웹사이트 아이디) 속성 : 
하나의 HTML 문서 내에서 
각각의 HTML 요소(태그)를 구분하는 식별값
->중복되는 id 가 존재하면 안된다!
