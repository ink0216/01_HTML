radio, checkbox

    - 여러 연관된 값을 묶어서 선택하는 경우에 사용함(이 중의 하나 또는 원하는 모든 것)

    -radio (button) : 여러 값 중 하나만 선택하는 경우 사용
     예시: 성별,  
    
    -check box : 여러 값 중 원하는 모든 것을 선택하는 경우
    예시: 상세 검색, 관심 분야 선택, 취미 선택 (여러 가지 중 원하는 것 여러개 선택)
    -둘은 용도는 다른데 사용 방법은 거의 동일
    -
        연관된 radio, checkbox는 같은 name 속성 값을 가져야 한다 !

label 태그들

    -입력 관련 태그(input, select, textarea 등)의 제목이나 설명을 나타내는 태그임
    -사용법 1 : label 태그 내부에 입력 관련 태그를 하나 작성하면 
                label 태그를 클릭해도 입력 관련 태그가 선택된다. 

    -사용법 2 : 입력 관련 태그의 id 속성값을 
                label 태그의 for 속성 값에 똑같이 적으면 연결된다.

select / option 태그 (둘은 세트임) 

    -select : 클릭하면 밑으로 메뉴가 촤르륵 내려오는 입력 태그(=드롭 다운)

    -option : select 클릭 시 보여질 메뉴를 작성하는 태그 (select에서 옵션을 하나 선택하겠다)

    *주의 사항 : select와 option은 
    value가 분리된 태그로 생각하면 된다!

textarea

    -여러 줄(row)을 입력 할 수 있는 입력 상자

    [속성] 
    -name : 제출되는 곳에서 데이터를 읽을 때(부를 때) 필요한 이름
    -rows : textarea의 행 크기를 지정하는 속성(rows="5"->5행 만큼 생김, 세로로 쓸 수 있는 글자 수가 5개)
    -cols : textarea의 열 크기를 지정하는 속성(cols="50"->50열 만큼 생김, 가로로 쓸 수 있는 글자 수가 50개)
    -style : "resize:none" : 크기 변경을 불가하게 만드는 것

    *textarea는 pre 태그처럼 그 안에 작성된 모양 그대로 출력하는 성질을 지님 아래에서 이름, 나이, 주소 앞에 빈 칸 생김( 쓴 그대로)
    공백 없었으면 좋겠으면 왼쪽으로 당겨 쓰면 됨

기타 입력 관련 태그

    - type="color" : 색을 선택할 수 있는 입력 상자를 출력함

    - type="file" : 파일을 선택할 수 있는 탐색기를 실행시킴

    - type="hidden"(숨겨져있는) : 화면에 보이지 않게 만드는 것
      (body 태그 안에 쓰면 화면에 다 나오는데, 주민번호 등 화면에 안보였으면 하는, 숨기고픈 것이 있을 때 사용함)


fieldset / legend

    field(넓은 영역) set(설정) : 테두리를 만들어서 영역을 설정하는 태그
                                주로 form 태그 내부에서 입력 요소의 그룹을 지어 구분할 때 사용함
                                회원가입 시 필수사항과 선택영역을 구분하는 테두리 만드는 경우 등 
    legend(제목) : fieldset 테두리에 이름을 지정하는 태그 (선 중간에 나옴 집합처럼)

button 태그

    -버튼을 만드는 태그

    -type 속성 값에 따라서 동작이 달라짐 (세 종류 존재)

    type="submit" : 제출 버튼 만들 때 사용, 
                    이게 button 태그의 기본값임->type 속성을 작성하지 않으면 자동으로 submit으로 지정됨
    
    type="reset" : 버튼이 포함된 form태그의 모든 입력값을 초기화는 버튼(이걸 누르면 모든 값들이 다 지워짐)
    
    type="button" : 아무런 기능이 없는 버튼(버튼 태그의 버튼 타입)
                    ->JS에서 기능을 추가하는 용도의 버튼(JS에서 여러 기능 만들어 이 버튼 눌렀을 때 그 기능 실행하도록 할 수 있음)
