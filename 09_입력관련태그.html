form 태그/ input 태그

form 태그(여러가지를 작성하고, 제출 누르면 구글폼처럼 내용이 전달됨)

-입력 양식을 만들기 위해 작성하는 판(영역)
- 내부에 작성된 입력 값을 지정된 곳으로 제출하는 역할을 함

[속성]
중요1) action 속성 : 제출될 곳을 지정하는 속성(보통 여기에 주소(인터넷 주소 등)를 작성)
중요2) method 속성(방식) : 설정된 곳으로 데이터를 전달하는 방식(제출을 이메일로 할지 우편으로 할지 방식을 정하는 것)
3) name 속성 : 이름 지정하는 속성 (JS(javascript에서 사용))
4) target 속성 : 제출된 페이지를 현재 탭에서 열 것인지 새 탭으로 열 것인지

input(입력) 태그


-웹 문서에서 사용자(클라이언트)가 입력할 수 있는 
양식을 제공하는 태그
-많은 type이 존재하고, input의 변형 형태(역할은 같지만 다른 종류의 태그들)도 다수 존재함
*input 태그에는 name 속성이 필수로 작성되어야 한다(꼭은 아니지만 80-90% 이상 작성하는 경우가 많음) !

숫자 관련 input 태그(숫자만 입력 가능)

    type="number" : 숫자만 입력 가능한 입력 상자를 만듦
    type="range" : 슬라이드바를 이용해서 숫자를 지정할 수 있게 만듦
         (단독 사용되지 않고, JS와 혼용 사용)

    [속성]
    1. min : 최소값
    2. max : 최대값
    3. step : 숫자가 증가/감소하는 간격(단계)를 조정
    4. value: 초기값(기본값)을 지정
